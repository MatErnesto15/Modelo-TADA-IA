Este repositorio contiene archivos modificados de otro archivo con el nombre original "mod1_ses5_distribuciones_muestrales.ipynb" (creado por el Dr. Ernesto Álvarez González).
Este archivo creado desde Google Colab, contiene teoría de distribuciones muestrales, y más particularmente sobre el teorema del límite central, con ejemplos 
programados con código python (gráficas, acceso a funciones estadísticas). También contiene ejercicios para el estudiante. La intención del archivo es evaluar el nivel
de comprensión de un tema central en estadística, generalmente difícil para estudiantes de contaduría pública y de administración de la Universidad Autónoma Benito Juárez de Oaxaca.
La hipótesis es que los estudiantes tienen mayores posibilidades de comprender dichos temas, al mismo tiempo que ganan libertad de gestionar su propio conocimiento:
si hay una duda, preguntan directamente a un chat inteligente. Finalmente, creemos que esta manera de plantear una clase les permite a los estudiantes ganar habilidades
para su vida profesional futura, pues la información se encuentra en archivos digitales y aprenden a darle un uso didáctico a los chats inteligentes, como Chat GPT o Gemini.
